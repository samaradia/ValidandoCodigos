# Validador de Código de Referência

Projeto simples em Java para validar códigos de referência utilizando Regex.

## Formato aceito

```text
ABC-1234
```

## Regras

- 3 letras maiúsculas
- 1 hífen (-)
- 4 números

## Exemplos

Código válido:

```text
ABC-1234
```

Código inválido:

```text
abc-1234
AB-1234
ABC1234
ABC-12
```

## Tecnologias

- Java
- Regex (Expressões Regulares)v
